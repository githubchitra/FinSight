# Implementation Plan - FinBot: AI Stock Market Analysis & Trading Assistant

Building a premium, comprehensive financial dashboard and AI assistant.

## Features & Modules

1.  **Educational Chatbot**: Context-aware AI assistant for financial concepts.
2.  **Stock Profile**: Real-time data, company overview, and key metrics.
3.  **Technical Analysis Engine**: Indicators (SMA, RSI, MACD) and Signal Generation.
4.  **Strategy Engine**: Multiple trading strategies with explanations.
5.  **Fundamental Analysis**: Valuation metrics and labels.
6.  **Sentiment Module**: News and Social Media sentiment scoring.
7.  **Signal Aggregator**: Final Buy/Sell recommendation with confidence.
8.  **Portfolio Simulator**: Virtual trading and P/L tracking.
9.  **Backtesting Engine**: Historical performance metrics and equity curves.
10. **AI Explanation Layer**: Natural language summaries of signals and data.

## Tech Stack

*   **Frontend**: React (via Vite)
*   **Styling**: Vanilla CSS (Premium aesthetics: Glassmorphism, Gradients, Lucide-react icons)
*   **Charts**: Lightweight Charts or Recharts
*   **Icons**: Lucide-react
*   **State Management**: React Context or Hooks
